# Project Dualis - Unity Frontend

This directory will contain the Unity 3D desktop client for Project Dualis.

## Status: Pending (Phase 3)

### Planned Features
- VRM/VTuber model loading
- Real-time lip-sync and facial expressions
- WebSocket IPC with Python backend
- STT/TTS integration (Whisper, Edge-TTS)
- Transparent window/overlay for desktop presence

### Technology Stack
- Unity 2022.3+ (LTS)
- UniVRM for model loading
- uLipSync for lip-sync
- WebSocketSharp for IPC
